## Getting started with Devbox

[Devbox](https://github.com/jetify-com/devbox) pins the full local toolchain (Node 20, pnpm 8, Rust via rustup, Solana CLI, Foundry, jq) so every developer and CI run uses identical versions.

**Prerequisites:** Install Devbox once:

```sh
curl -fsSL https://get.jetify.com/devbox | bash
```

**Enter the dev shell:**

```sh
devbox shell
```

All pinned tools (node, pnpm, cargo, solana, forge, cast, anvil, jq) are on PATH inside the shell. Exit with `exit` or Ctrl-D.

**Run project scripts via Devbox:**

```sh
devbox run build   # pnpm -r run build
devbox run test    # pnpm -r test --if-present
devbox run lint    # pnpm lint
```

The `devbox.lock` file (generated by `devbox install`) pins exact Nix package versions — commit it after the first local install so teammates and CI use bit-for-bit identical packages.
